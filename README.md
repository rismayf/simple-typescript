Basic Typescript
===================

Content
-------
### Repository Anatomy

- `dist` - output folder where the compiled ts code stored
- `src` - source code used on training

Getting Started
---------------

``` bash
npm install
npm run build
npm start
```
Make sure you include folder of the compiled code at `.gitignore`
